## Tips

- Use `Ctrl + Shift + P` to access any command if you don't remember the shortcut
- Most multi-key shortcuts use `Ctrl + K` as a prefix
- You can customize all shortcuts in Settings â†’ Keyboard Shortcuts
- Hover over shortcuts in the command palette to see them displayed
