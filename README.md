# Learn PlayWright Batch 2x

This repository contains a structured JavaScript learning course with practice files organized by chapter. It is designed for beginners who want to build a strong foundation in JavaScript concepts step by step.

## Course Overview

The lessons cover:
- JavaScript basics and setup
- Variables, functions, scope, and hoisting
- Identifiers, literals, and comments
- Operators and control flow
- Conditional logic with if/else statements
- Switch-statement concepts and practice
- User input handling and loops

## Getting Started

1. Clone this repository.
2. Open the files in your editor.
3. Follow the lessons in order by chapter.
4. Run JavaScript files with Node.js:
   node filename.js

## Prerequisites

- Node.js installed on your system
- A code editor such as VS Code
- Basic familiarity with the terminal

## How to Use

Each file contains JavaScript code and comments explaining the concept. Follow the numeric order within each chapter to progress smoothly.

- Basics: setup, commands, and fundamentals
