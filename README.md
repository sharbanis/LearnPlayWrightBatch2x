- 15_letblockscope.js - Block scope with let

## Getting Started

1. Clone this repository
2. Open the files in your favorite code editor
3. Follow the lessons in order by chapter
4. Run each file using Node.js: 
ode filename.js

## Prerequisites

- Node.js installed on your system
- Basic text editor or IDE (VS Code recommended)
- Familiarity with command line basics

## How to Use

Each file contains JavaScript code with comments explaining concepts. Follow the numeric order within each chapter to progress through the material.

## Topics Covered

- **Basics**: Setup, commands, and fundamentals
- **JavaScript Concepts**: Variables, functions, scope, and hoisting
- **Identifiers & Comments**: Naming rules and best practices
- **Advanced Concepts**: Temporal Dead Zone (TDZ), variable hoisting, block scope
- **Literals**: Numbers, strings, templates, null, undefined, and special values
- **Operators**: Arithmetic, comparison, logical, assignment, and special operators
